# Data Dictionary

Dataset files:

- `data/ecommerce_pricing_data.csv`
- `data/ecommerce_pricing_data.json`

The dataset is synthetic, reproducible, and generated by `data/generate_dataset.js`. It mimics an e-commerce catalog with controlled price tests, promotional periods, traffic variation, and conversion response by product.

| Column | Type | Description |
| --- | --- | --- |
| `product_id` | string | Stable product identifier. |
| `product_name` | string | Human-readable synthetic product name used in the dashboard. |
| `price` | number | Selling price observed for the product on the given date, in USD. |
| `units_sold` | integer | Units sold on the given date. |
| `page_views` | integer | Product detail page views on the given date. |
| `conversion_rate` | number | `units_sold / page_views`, rounded to four decimals. |
| `product_category` | string | Product category such as Electronics, Beauty, or Grocery. |
| `date` | date string | Observation date in `YYYY-MM-DD` format. |

Synthetic generation notes:

- Prices vary through promotion windows, premium tests, and day-level noise.
- Conversion rates follow a product-specific price elasticity curve plus seasonality and noise.
- Page views vary independently from price to avoid baking in a perfect relationship.
- The date range is January 1, 2025 through June 30, 2025.
